# Data Cleaning & Visualization using Python

## 📌 Project Overview

This project demonstrates the complete Data Cleaning and Exploratory Data Analysis (EDA) process using Python. The dataset was cleaned, analyzed, and visualized to extract meaningful business insights.

---

## 🎯 Objectives

- Clean raw data
- Handle missing values
- Remove duplicate records
- Detect and treat outliers
- Perform Exploratory Data Analysis (EDA)
- Generate business insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Visualizations

- Sales Distribution
- Profit Distribution
- Category Count
- Region-wise Sales
- Region-wise Profit
- Payment Method Distribution
- Customer Type Distribution
- Quantity Distribution
- Sales vs Profit
- Correlation Heatmap
- Monthly Sales Trend
- Top 10 Cities by Sales
- Profit by Category
- Sales Boxplot
- Pairplot

---

## 📈 Business Insights

- Sales vary significantly across transactions.
- Profit differs by product category.
- Some cities generate higher sales than others.
- Sales and Profit show a positive relationship.
- Regional performance varies considerably.
